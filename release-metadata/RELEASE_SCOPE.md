# Release Scope

This source release is intended to preserve all OpenAMRobot-owned technical material needed to inspect, reproduce, assemble, flash, bring up, operate, maintain, and continue developing the platform within the limits documented for v0.0.1.

## Included

- Hardware and manufacturing sources
- ROS 2 software and simulation
- Embedded firmware sources
- User-interface sources and assets
- Interfaces and communication definitions
- Documentation and governance files
- Component provenance, file inventory, and checksums

## Not yet guaranteed

- Fully disconnected installation of Ubuntu, ROS, Docker, toolchains, and third-party packages
- Certified industrial safety
- Guaranteed compatibility with every supplier revision or substituted component
- Production reliability without user validation
